Projede kullanılan model dosyaları (model.safetensors, config, tokenizer, tokenizer_config)
yükleme limiti nedeniyle zip dosyasına eklenmemiştir.

Bu dosyalar, train.py çalıştırılarak yeniden oluşturulabilir
veya kullanılan Hugging Face modeli tekrar indirilebilir.

Projeyi çalıştırmak için gerekli ana kaynak kodları:
- app.py
- train.py
- data.json
dosyaları zip içerisinde yer almaktadır.
